About Hoodie

Hoodie enables you to express yourself through technology by making web application development very fast, easy and accessible.

Hoodie is a free and Open Source Software for building applications for the web and iOS. It is a complete backend for your apps, ready for you to get creative. It works immediately out-of-the-box: develop your frontend code, plug it into Hoodieâ€™s frontend-friendly API and your app is ready.

Principles

Hoodie is built on three core principles:

  • Offline First: Hoodie considers the offline case first, and allows you to use your apps offline, with no extra code.
  • noBackend: Hoodie abstracts the backend away into an expressive, accessible API accessed entirely within your frontend code.
  • Framework Agnostic: Hoodie doesn't rely on a particular frontend framework, so you're free to choose one that fits your needs, or none at all!

Docs & Support

The easiest way to really get started with Hoodie is to check out the full docs and info at www.hood.ie. Hoodie is also open-source and has a community on Github, so check that out if you want to report bugs or create a pull request.
